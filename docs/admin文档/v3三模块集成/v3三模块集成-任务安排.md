### Task B：迁入 v3 UI
- 迁入 `CustomerManager`

- 替换页面 localStorage / mock 数据依赖
- 接入 loading / submit / refetch

### Task D：入库管理数据层替换
- 新增 goods 前后端接口
- 替换页面本地数据依赖
- 接入客户 / 员工真实选择源

### Task E：费用管理数据层替换
- 新增 expenses 前后端接口
- 替换页面本地数据依赖
- 接入审批流与提交反馈

### Task F：回归验证
- 构建 admin
- 启动 admin
- 验证三个新模块页面与关键操作
- 回归旧模块
