## 2. 模块定位

考勤计算管理模块负责将员工原始考勤记录转换成可用于薪资核算的考勤结果。

模块负责：

- 读取员工考勤记录。
- 读取员工在考勤日期对应的考勤规则。
- 计算每日原始工时。
- 计算每日休息扣除时长。
- 计算每日有效工时。
- 计算每日标准工时。
- 计算每日加班原始时长。
- 计算每日加班计薪时长。
- 汇总自然月考勤结果。
- 标记考勤计算异常。
- 给薪资核算模块提供自然月考勤输入。

模块不负责：

- 员工档案维护。
- 员工状态维护。
- 考勤规则配置。
- 员工绑定考勤规则。
- 一次性薪资项维护。
- 最终工资计算。

### 3.3 当前计算结果字段

当前代码中已有 `AttendanceDetails`：

```ts
export interface AttendanceDetails {
  raw: number;
  valid: number;
  standard: number;
  ot: number;
}
```

字段含义：

- `raw`：原始工时。
- `valid`：有效工时，当前等于原始工时减去休息重叠时长。
- `standard`：标准工时，来自 `config.standardHours`。
- `ot`：加班时长，当前等于 `valid - standardHours`，小于 0 时为 0。

### 3.4 当前计算函数

当前计算函数：

```ts
export function calcAttendanceDetails(rec: AttendanceRecord, config: AppConfig): AttendanceDetails
```

当前计算逻辑：

- 如果没有上班时间、没有下班时间、类型为缺勤、类型为假期，则返回 0 工时。
- 上班时间、下班时间通过 `parseTimeToHours` 转成小时数。
- 如果下班时间小于上班时间，则按跨天处理，下班时间加 24 小时。
- 原始工时：`raw = outTime - inTime`。
- 休息扣除：根据 `breakStart` / `breakEnd` 与实际打卡时间的重叠区间扣除。
- 有效工时：`valid = raw - breakDeduction`。
- 加班时长：`ot = valid - standardHours`，小于 0 时为 0。

## 8. 计算规则梳理

### 8.1 匹配考勤规则

每日计算必须先匹配考勤规则：

1. 根据员工 ID 查询员工考勤规则历史。
2. 根据考勤日期匹配当日有效的员工规则历史。
3. 读取对应 `AttendanceRule`。
4. 检查考勤日期是否在规则自身有效期内。
5. 如果找不到有效规则，生成异常结果，不静默计算。

### 8.2 原始工时

计算方式：

```text
原始工时 = 下班时间 - 上班时间
```

跨天规则：

```text
如果下班时间小于上班时间，则下班时间按次日处理，加 24 小时。
```

### 8.3 休息扣除

计算方式：

```text
休息扣除时长 = 实际打卡区间 与 考勤规则休息区间 的重叠时长
```

注意：

- 当前代码已经按重叠区间扣除。
- 后续结果中需要把该值作为字段返回，便于核对。

### 8.4 有效工时

计算方式：

```text
有效工时 = max(0, 原始工时 - 休息扣除时长)
```

### 8.5 加班原始时长

计算方式：

```text
加班原始时长 = max(0, 有效工时 - 标准工时)
```

### 8.6 加班计薪时长

已确认规则：按 0.5 小时向下取整。

计算方式：

```text
加班计薪时长 = Math.floor(加班原始时长 / 0.5) * 0.5
```

如果考勤规则关闭加班：

```text
加班原始时长可以保留展示，加班计薪时长为 0。
```

### 8.7 缺勤和假期

当前处理口径：

- 原始记录不存在时，前端当前展示为缺勤。
- 员工状态为休假时，前端当前展示为假期。
- 本轮不做复杂缺勤、迟到、早退自动规则。
- 缺勤和假期工时均为 0。

建议支持：

- 重新计算单日员工考勤
- 重新计算某员工某月考勤
- 重新计算全部员工某月考勤

触发场景：

- 原始考勤记录调整后
- 员工考勤规则切换后
- 考勤规则被编辑后

注意：考勤规则允许被直接编辑，因此历史重算可能改变历史结果。

## 9.6 异常处理

需要标记异常，不应静默跳过。

异常场景：

- 找不到员工。
- 找不到考勤日期对应的有效考勤规则。
- 上班时间缺失。
- 下班时间缺失。
- 时间格式错误。
- 生效规则已过期。

异常结果可保留在每日结果中，并阻止进入薪资核算。

- 休息结束时间

- 加班取整方式

不负责编辑考勤规则。

考勤计算输出：

- 自然月总有效工时
- 自然月总标准工时
- 自然月总加班计薪时长
- 是否可进入薪资核算

薪资核算读取这些结果，不应再直接遍历原始考勤记录自行计算。
